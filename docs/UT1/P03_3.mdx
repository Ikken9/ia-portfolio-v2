## Objetivos
- Comparar Lineal Regression vs. Logistic Regression

### Comparación
| Aspecto           |           Regresión Lineal            |         Regresión Logística          |
|:------------------|:-------------------------------------:|:------------------------------------:|
| Qué predice       | Valores continuos (números continuos) |    Clases/categorías (categorías)    |
| Ejemplo de uso    |          Precio de viviendas          | Diagnóstico médico (benigno/maligno) |
| Rango de salida   |    -∞ a +∞ (cualquier número real)    |        0 a 1 (probabilidades)        |
| Métrica principal |   RMSE/R² (Error cuadrático medio)    |    Exactitud/Precisión/Recall/F1     |


### ¿Cuál modelo usarías para predecir el salario de un empleado?
Usaría regresión lineal porque el salario es un valor numérico continuo.

### ¿Cuál modelo usarías para predecir si un email es spam?
Usaría regresión logística porque es un problema de clasificación binaria (solo dos categorías: spam o no spam).

### ¿Por qué es importante separar datos de entrenamiento y prueba?
Es crucial para evaluar la capacidad de generalización del modelo. Si usáramos todos los datos para entrenar, no
podríamos saber si el modelo realmente aprendió patrones generales o simplemente hizo un *overfit*.
